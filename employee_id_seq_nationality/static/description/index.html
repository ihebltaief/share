Employee Id By Nationality

Generate automatic id for employees with nationality code prefix when create or import employees.

Key features
  • Generate automatic sequence employee id with country code prefix
  • Support odoo version 13 12 11 10 .
Module Details

Operation steps

When create employees you should select Nationality

Identification Id will be created automatically with country prefix, it will appear in Tree View

New group filter "Nationality"

New group filter "Nationality"

Identification Id appear in kanban view also

Notes

  1. To Modify Sequence Options

    If you want to modify the format of the sequence, go to Settings -> Technical -> Sequences & Identifiers -> Sequences and search for the "Employee Identifier By Nationality" sequence, where you modify numbering formats...

  2. When installing the module, the ID of existing employees is not generated automatically
