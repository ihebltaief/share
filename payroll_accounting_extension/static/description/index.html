Payroll Accounting Extension

Journal Entry lines linked with partner and analytic account linked only with expense accounts

Key features
  • Choose the rule that you want to be linked with partner like Loan and Salary advance
  • Analytic account configured in rules will be linked only with expense accounts not with all journal entry lines like standard Odoo
Module Details

Operation steps

Modify you rule and configure debit and credit accounts (Only account with type receivable or payable can affect the partner)

Check the field "Affect Partner"

Employee should be linked with contact(partner)

Select The analytic account in the running contract

Works with single payslip

Works with batch payslip

Notes

  1. This module works with entreprise edition
